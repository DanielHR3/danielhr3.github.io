## 🎯 Sobre el portfolio

Portfolio personal diseñado para presentar mi trabajo, habilidades y trayectoria profesional. Sin frameworks ni dependencias pesadas — construido con HTML, CSS y JavaScript puro para máxima velocidad y control.

## 📄 Secciones

| Sección | Contenido |
|---|---|
| **Hero** | Foto, nombre, badge de disponibilidad y CTAs |
| **Skills** | Stack tecnológico en pills interactivos con hover |
| **Proyectos** | Cards glassmorphism con 5 proyectos destacados |
| **Experiencia** | Timeline vertical de roles y formación |
| **Contacto** | Footer con LinkedIn, CV y GitHub |

## 🛠️ Tecnologías

- **HTML5** — Estructura semántica
- **CSS3** — Variables custom, animaciones, glassmorphism, grid
- **Tailwind CDN** — Utilidades de layout
- **AOS.js** — Animaciones on-scroll
- **Google Fonts** — Space Grotesk + Inter
- **GitHub Pages** — Hosting gratuito

## 🎨 Diseño

- Paleta: `#0a0a0a` (fondo) + `#6B8E23` (accent verde olivo) + `#A67B5B` (warm)
- Tipografía: **Space Grotesk** (headings) + **Inter** (body)
- Efectos: glassmorphism en cards, grid background, glow en hover, loader animado
